Cómo realizar AA con TensorFlow

Descripción general

En este lab, continuaremos con el notebook de Datalab del lab anterior para crear una red neuronal de TensorFlow capaz de predecir la demanda de taxis en una ciudad.

Qué aprenderá

En este lab, aprenderá lo siguiente:

Cómo usar TensorFlow para crear un modelo de red neuronal capaz de predecir la demanda de taxis en Nueva York

Introducción

En este lab, continuará creando un sistema de previsión de la demanda mediante aprendizaje automático. En el lab anterior, creó un conjunto de datos de entrenamiento con los factores que afectan la demanda de taxis (“atributos de entrada”) y la demanda real que hubo (el “objetivo”) durante un período anterior compuesto de varios días. En este lab, usará estos datos para entrenar una red neuronal de TensorFlow capaz de predecir la demanda diaria en el futuro a partir de esos atributos de entrada.

Configuración

Tarea 1: Inicie Cloud Datalab

Siga estos pasos para iniciar Cloud Datalab:

  1. En Cloud Shell, ingrese lo siguiente:
datalab create bdmlvm --zone us-central1-a

Datalab tardará unos 5 minutos en iniciarse.

Nota: Recuerde seguir las indicaciones durante el proceso.

Tarea 2: Registre el notebook en Cloud Datalab

Si es necesario, espere hasta que Datalab termine de iniciarse. Estará listo cuando vea un mensaje que le sugiera usar una “Web Preview”.

  1. Haga clic en el ícono de Vista Previa Web, ubicado en la esquina superior derecha de la cinta de Cloud Shell. Haga clic en Cambiar puerto. Cambie al puerto 8081 con el cuadro de diálogo Cambiar Puerto de Vista Previa y, luego, haga clic en Cambiar Y Obtener Vista Previa.

    Nota: La conexión a su instancia de Datalab se mantiene abierta mientras permanezca activo el comando datalab. Si se cierra o interrumpe la instancia de Cloud Shell utilizada para ejecutar el comando datalab, perderá la conexión con su VM de Cloud Datalab. Si esto ocurre, tal vez pueda restablecer la conexión mediante el comando datalab connect bdmlvm en una nueva instancia de Cloud Shell.

  2. En Datalab, haga clic en el ícono Open ungit ubicado en la esquina superior derecha.

  3. En la ventana de Ungit, seleccione el texto que dice /content/datalab/notebooks, quite la palabra notebooks para que quede /content/datalab y presione la tecla Enter.

  4. En el panel que aparecerá, escriba lo siguiente para indicar el repositorio de GitHub que Clone from:

    https://github.com/GoogleCloudPlatform/training-data-analyst

  5. Haga clic en Clone repository.

Tarea 3: Abra un notebook de Datalab

  1. En el navegador de Datalab, vaya a training-data-analyst > CPB100 > lab4a > demandforecast.ipynb.

  2. Igual que en el lab anterior, ejecute todas las celdas del notebook hasta llegar a la sección Machine Learning with Tensorflow.

  3. Siga ejecutando las celdas restantes.

  4. Lea los comentarios y haga clic en Clear | Clear all cells. Luego, ejecute los fragmentos de Python en la celda, paso a paso (use Shift+Enter para ejecutar cada parte del código).
