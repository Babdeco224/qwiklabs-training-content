ML mit TensorFlow ausführen

Übersicht

In diesem Lab setzen Sie das Datalab-Notebook aus dem vorherigen Lab fort, um ein neuronales TensorFlow-Netzwerk zur Ermittlung des Taxibedarfs zu erstellen.

Lerninhalte

Aufgaben in diesem Lab:

TensorFlow nutzen, um ein Modell für ein neuronales Netzwerk zu erstellen, mit dem der Taxibedarf in New York prognostiziert wird

Einführung

In diesem Lab setzen Sie Ihre Arbeit an einem System für die Bedarfsprognose fort und nutzen hierfür maschinelles Lernen. Im vorherigen Lab haben Sie ein Trainings-Dataset erstellt, das die Faktoren, die den Taxibedarf beeinflussen ("Eingabemerkmale"), und den tatsächlichen Bedarf ("Ziel") der vergangenen Tage umfasst. In diesem Lab verwenden Sie diese Daten, um ein neuronales TensorFlow-Netzwerk zu trainieren, mit dem sich anschließend auf Grundlage dieser Eingabemerkmale der Bedarf der kommenden Tage voraussagen lässt.

Einrichtung

Aufgabe 1: Cloud Datalab starten

So starten Sie Cloud Datalab:

  1. Geben Sie Folgendes in Cloud Shell ein:
datalab create bdmlvm --zone us-central1-a

Das Starten von Datalab dauert etwa 5 Minuten.

Hinweis: Folgen Sie bei diesem Vorgang den Eingabeaufforderungen.

Aufgabe 2: Notebook in Cloud Datalab auschecken

Warten Sie gegebenenfalls, bis Datalab hochgefahren ist. Datalab ist bereit, wenn eine Nachricht angezeigt wird, die Sie zu einer Webvorschau auffordert.

  1. Klicken Sie oben rechts in der Cloud Shell-Seitenleiste auf das Symbol Webvorschau. Klicken Sie auf Port ändern. Ändern Sie den Port über das Dialogfeld Vorschauport ändern in 8081 und klicken Sie dann auf Ändern und Vorschau.

    Hinweis: Die Verbindung zur Datalab-Instanz bleibt so lange bestehen, wie der Befehl "datalab" aktiv ist. Wenn die zum Ausführen des Befehls "datalab" verwendete Cloud Shell-Anwendung geschlossen oder unterbrochen wurde, wird die Verbindung zu Ihrer Cloud Datalab-VM beendet. In diesem Fall können Sie die Verbindung in einer neuen Cloud Shell-Anwendung mit dem Befehl datalab connect bdmlvm wiederherstellen.

  2. Klicken Sie in Datalab auf der oberen rechten Seitenleiste auf das Symbol Open ungit.

  3. Markieren Sie im Ungit-Fenster den Text /content/datalab/notebooks und löschen Sie notebooks, sodass dort nur noch /content/datalab steht. Drücken Sie anschließend die Enter.

  4. Geben Sie im neu angezeigten Feld als GitHub-Repository für den Clone from Folgendes ein:

    https://github.com/GoogleCloudPlatform/training-data-analyst

  5. Klicken Sie auf Clone repository.

Aufgabe 3: Datalab-Notebook öffnen

  1. Gehen Sie im Datalab-Browser zu training-data-analyst > CPB100 > lab4a > demandforecast.ipynb

  2. Führen Sie wie beim vorherigen Lab jede Zelle im Notebook aus, bis Sie den Abschnitt Machine Learning with TensorFlow erreichen.

  3. Führen Sie die verbleibenden Zellen aus.

  4. Lesen Sie die Bemerkungen. Klicken Sie dann auf Clear | Clear all Cells und führen Sie Schritt für Schritt die Python-Snippets in den Zellen aus. Verwenden Sie die Tastenkombination Shift+Enter, um alle Teile des Codes auszuführen.
