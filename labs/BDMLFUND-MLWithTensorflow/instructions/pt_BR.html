Usar machine learning com o Tensorflow

Visão geral

Neste laboratório, você verá como usar conjuntos de dados do Datalab do laboratório anterior para criar uma rede neural do TensorFlow e prever a demanda de táxis.

Conteúdo do laboratório

Neste laboratório, você aprenderá a:

usar o TensorFlow para criar um modelo de rede neural e fazer previsões da demanda de táxis em Nova York

Introdução

Neste laboratório, você também verá como criar um sistema de previsão de demanda usando o machine learning. No laboratório anterior, você criou um conjunto de dados de treinamento com base em fatores que afetam a demanda ("recursos de entrada") e na demanda real ("meta") dos dias que já passaram. Agora, você usará esses dados para treinar uma rede neural do TensorFlow que poderá prever a demanda para os próximos dias de acordo com os recursos de entrada.

Configurar

Tarefa 1: inicie o Cloud Datalab

Para iniciar o Cloud Datalab, faça o seguinte:

  1. No Cloud Shell, digite o código:
datalab create bdmlvm --zone us-central1-a

O Datalab levará cerca de cinco minutos para inicializar.

Observação: siga as instruções na tela durante esse processo.

Tarefa 2: insira um conjunto de dados no Cloud Datalab

Se necessário, espere até que o Datalab inicialize. A ferramenta estará pronta quando aparecer a mensagem "Web Preview".

  1. Clique no ícone Visualização da Web no canto superior direito da barra lateral de opções do Cloud Shell. Clique em Alterar porta. Alterne para a porta 8081 na caixa de diálogo Alterar porta de visualização. Em seguida, clique em Alterar e Visualizar.

    Observação: a conexão com sua instância do Datalab continuará aberta enquanto o comando estiver ativo. Se o Cloud Shell usado para executar o comando datalab for fechado ou interrompido, a conexão com sua VM do Cloud Datalab será encerrada. Caso isso aconteça, use o comando datalab connect bdmlvm no seu novo Cloud Shell.

  2. No Datalab, clique no ícone Open ungit na barra lateral do canto superior direito.

  3. Na janela do Ungit, selecione o texto /content/datalab/notebooks e remova notebooks para que fique apenas /content/datalab. Em seguida, pressione Enter.

  4. No painel exibido, digite o endereço a seguir como o repositório do GitHub em Clone from:

    https://github.com/GoogleCloudPlatform/training-data-analyst

  5. Clique em Clone repository.

Tarefa 3: abra um conjunto de dados do Datalab

  1. No navegador do Datalab, acesse training-data-analyst > CPB100 > lab4a > demandforecast.ipynb.

  2. Assim como no laboratório anterior, execute cada célula no conjunto de dados até chegar à seção Machine Learning with Tensorflow.

  3. Continue a executar as células restantes.

  4. Leia o comentário, clique em Clear | Clear all Cells e execute os snippets do Python na célula. Pressione Shift+Enter para executar cada trecho do código.
