APIs für maschinelles Lernen aufrufen

Übersicht

In diesem Lab nutzen Sie APIs für maschinelles Lernen in Datalab.

Lerninhalte

In diesem Lab lernen Sie, wie Sie mit Python APIs für maschinelles Lernen aufrufen und deren Ergebnisse verwenden.

Einführung

In diesem Lab rufen Sie die folgenden APIs für maschinelles Lernen aus Datalab auf, um bestimmte Aufgaben auszuführen:

  • Vision API zur Erkennung von Text in einem Bild

  • Translate API zum Übersetzen des Texts ins Englische

  • Natural Language API für die Sentimentanalyse einiger berühmter Zitate

  • Speech API für das Transkribieren einer Audiodatei

Einrichtung

Aufgabe 1: Cloud Datalab starten

So starten Sie Cloud Datalab:

  1. Geben Sie Folgendes in Cloud Shell ein:

datalab create bdmlvm --zone us-central1-a

Das Starten von Datalab dauert etwa 5 Minuten.

Hinweis: Folgen Sie bei diesem Vorgang den Eingabeaufforderungen.

Aufgabe 2: Notebook in Cloud Datalab auschecken

Warten Sie gegebenenfalls, bis Datalab hochgefahren ist. Datalab ist bereit, wenn eine Meldung angezeigt wird, die Sie zu einer Webvorschau auffordert.

  1. Klicken Sie oben rechts in der Cloud Shell-Seitenleiste auf das Symbol Webvorschau. Klicken Sie auf Port ändern. Ändern Sie den Port über das Dialogfeld Vorschauport ändern in 8081 und klicken Sie dann auf Ändern und Vorschau.

    Hinweis: Die Verbindung zur Datalab-Instanz bleibt so lange bestehen, wie der Befehl "datalab" aktiv ist. Wenn die zum Ausführen des Befehls "datalab" verwendete Cloud Shell-Anwendung geschlossen oder unterbrochen wurde, wird die Verbindung zu Ihrer Cloud Datalab-VM beendet. In diesem Fall können Sie die Verbindung in einer neuen Cloud Shell-Anwendung mit dem Befehl datalab connect bdmlvm wiederherstellen.

  2. Klicken Sie in Datalab auf der oberen rechten Seitenleiste auf das Symbol Open ungit.

  3. Markieren Sie im Ungit-Fenster den Text /content/datalab/notebooks und löschen Sie notebooks, sodass dort nur noch /content/datalab steht. Drücken Sie anschließend die Enter.

  4. Geben Sie im neu angezeigten Feld als GitHub-Repository für den Clone from Folgendes ein:

    https://github.com/GoogleCloudPlatform/training-data-analyst

  5. Klicken Sie auf Clone repository.

Aufgabe 3: API-Schlüssel abrufen

So rufen Sie einen API-Schlüssel ab:

  1. Klicken Sie in der GCP Console im Navigationsmenü () auf APIs & Dienste und dann auf Bibliothek.

  2. Geben Sie im Suchfeld den Begriff vision ein, um die Cloud Vision API von Google zu finden. Klicken Sie dann auf den Hyperlink.

  3. Klicken Sie bei Bedarf auf Aktivieren.

  4. Aktivieren Sie anschließend auf die gleiche Weise die Cloud Translation API, die Cloud Speech API und die Cloud Natural Language API von Google.

  5. Wählen Sie im Menü der GCP Console APIs & Dienste und dann Anmeldedaten aus.

  6. Falls Sie noch keinen API-Schlüssel haben, klicken Sie auf die Schaltfläche Anmeldedaten erstellen und wählen Sie API-Schlüssel aus. Kopieren Sie den neu erstellten API-Schlüssel und klicken Sie dann auf Schließen. Diesen API-Schlüssel benötigen Sie später im Notebook.

Aufgabe 4: APIs für maschinelles Lernen in Datalab aufrufen

  1. Kehren Sie zum Datalab-Browser zurück und rufen Sie training-data-analyst > CPB100 > lab4c > mlapis.ipynb auf.

  2. Lesen Sie die Bemerkungen. Klicken Sie dann auf Clear | Clear all Cells und führen Sie Schritt für Schritt die Python-Snippets in den Zellen aus. Verwenden Sie die Tastenkombination Shift + Enter, um alle Teile des Codes auszuführen. Fügen Sie Ihren API-Schlüssel in die erste Python-Zelle ein.
