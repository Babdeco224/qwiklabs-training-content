Criar uma instância do Compute Engine

Visão geral

Neste laboratório, você aprenderá como provisionar uma máquina virtual, configurar a segurança dela e acessá-la remotamente.

Conteúdo do laboratório

Neste laboratório, você aprenderá a:

  • criar uma instância do Compute Engine com o acesso e a segurança necessários;

  • usar o SSH para acessar a instância.

Introdução

Neste laboratório, você aprenderá a provisionar uma máquina virtual, configurar o acesso à API e fazer login na VM de maneira remota. Você raramente trabalhará com instâncias do Compute Engine em um nível tão baixo. No entanto, entender as noções básicas do GCP pode ser útil para resolver problemas.

Tarefa 1: crie uma instância do Compute Engine com o acesso necessário à API

Para criar uma instância do Compute Engine, faça o seguinte:

  1. No Console do GCP, acesse o menu de navegação () e clique em Compute Engine.

  2. Clique em Criar e aguarde até que um formulário seja carregado. Será necessário alterar algumas opções no formulário que aparecer.

  3. Em Nome, deixe o valor padrão, em Região, selecione us-central1 e em Zona, selecione us-central1-a.

  4. Na opção Identidade e acesso à API, em Escopos de acesso, selecione Permitir acesso completo a todas as APIs do Cloud:

  5. Clique em Criar

Tarefa 2: use SSH na instância

Após a criação da instância, você poderá acessá-la remotamente do Compute Engine usando o Secure Shell (SSH):

  1. Quando a instância criada estiver disponível, clique em SSH:

    Observação: as chaves SSH são transferidas de modo automático para que você se conecte ao SSH diretamente do seu navegador, sem precisar de outro software.

  2. Para ver informações sobre a nova instância do Compute Engine, digite o seguinte código no seu terminal SSH:

    cat /proc/cpuinfo

Tarefa 3: instale o software

  1. No terminal SSH, digite o seguinte código:

    sudo apt-get update
sudo apt-get -y -qq install git

  2. Verifique se o git está instalado:

    git --version

  3. Digite o seguinte comando para sair da sessão:

    exit
