Créer une instance Compute Engine

Présentation

Dans cet atelier, vous allez apprendre à créer une machine virtuelle, configurer ses paramètres de sécurité et y accéder à distance.

Objectifs de l'atelier

Au cours de cet atelier, vous allez apprendre à réaliser les opérations suivantes :

  • Créer une instance Compute Engine et définir les paramètres d'accès et de sécurité appropriés

  • Établir une connexion SSH à l'instance

Introduction

Dans cet atelier, vous allez démarrer une machine virtuelle, configurer ses paramètres d'accès aux API et vous y connecter à distance. À ce niveau de compétence, vous n'aurez à utiliser des instances Compute Engine que rarement, mais il peut être utile de connaître les principes de base de GCP en cas de problème.

Tâche 1 : Créer une instance Compute Engine et définir ses paramètres d'accès aux API

Pour créer une instance Compute Engine, procédez comme suit :

  1. Dans la console GCP, accédez au menu de navigation (), puis cliquez sur Compute Engine.

  2. Cliquez sur Create (Créer), puis patientez pendant le chargement du formulaire. Vous allez devoir modifier certaines options sur celui-ci.

  3. Laissez la valeur par défaut dans le champ Name (Nom). Ensuite, sélectionnez us-central1 dans Region (Région) et us-central1-a dans Zone (Zone).

  4. Pour le paramètre Access scopes (Champs d'application de l'accès) du menu Identity and API access (Identité et accès à l'API), sélectionnez Allow full access to all Cloud APIs (Autoriser l'accès complet à l'ensemble des API Cloud).

  5. Cliquez sur Create (Créer).

Tâche 2 : Se connecter en SSH à l'instance

Une fois votre instance Compute Engine créée, vous pouvez y accéder à distance à l'aide du protocole Secure Shell (SSH) :

  1. Lorsque l'instance que vous venez de créer est disponible, cliquez sur SSH :

    Remarque : Les clés SSH sont transférées automatiquement. Vous pouvez ainsi vous connecter en SSH directement depuis votre navigateur, sans utiliser de logiciel spécifique.

  2. Pour obtenir des informations sur l'instance Compute Engine que vous venez de lancer, saisissez ce qui suit dans votre terminal SSH :

    cat /proc/cpuinfo

Tâche 3 : Installer le logiciel

  1. Dans le terminal SSH, saisissez la commande ci-dessous :

    sudo apt-get update
sudo apt-get -y -qq install git

  2. Vérifiez que le logiciel git est bien installé. Pour ce faire, exécutez la commande suivante :

    git --version

  3. Quittez la session en saisissant la commande ci-dessous :

    exit
